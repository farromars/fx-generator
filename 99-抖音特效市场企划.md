## 2. 三个品类深度评估（v0.1 候选）

### 2.1 品类 1：静态面部贴图（你当前的 S1）

**事实**：
- ✅ 制作门槛最低：1 张带 alpha 的 PNG 就能出
- ✅ AI（SD/SDXL）最擅长的形态
- ✅ 抖音 EC 也最支持

**问题**（PM 视角必须直说）：
- 🔵 **存量极高、同质化严重**：抖音里同主题（赛博/纹身/动漫脸）特效已有数百个
- 🔵 **审核更严**：因为太容易做，平台对原创性、是否过度美颜、是否构成"假脸误导"判定更严
- 🔵 **流量天花板低**：用户更喜欢"动起来"的特效（贴纸 / 全屏），静态面部贴图使用率排倒数
- ⚠️ **新人冷启动难**：抖音特效流量分发有"老创作者"权重，新号上架的静态贴图很可能 < 100 次使用

**结论**：**v0.1 跑通流程可以做，但不要把它当主战场**。出 1-2 个上架走通流程后立刻转 S2。

### 2.2 品类 2：动态贴纸 / 头饰（推荐 v0.2 主攻）

**事实**：
- ✅ 抖音特效流量大头（火焰皇冠、樱花头饰、节日帽 长期在热门榜）
- ✅ 序列帧是 EC 原生支持的格式（`frame_0001.png` ... `frame_0030.png`）
- ✅ AI 生成对此适合度高：**SD + AnimateDiff / SVD / EbSynth** 可以快速产 30 帧序列

**为什么更容易上架**：
- 🔵 同主题特效虽然多，但每个都有自己的"动态风格"，原创性更容易判定
- 🔵 用户使用率天然高（动起来好看 → 更多人拍 → 平台分发权重更高）
- 🔵 节日窗口（春节/七夕/中秋等）平台会主动推

**技术挑战**（v0.2 要解决的）：
- 需要"图生视频"或"图生序列帧"的 AI 能力（你目前工具只有 text2img）
- 序列帧每张都要带 alpha 通道（需要每帧都跑 rembg）
- 帧间一致性（同一主题 30 帧不能跳变）

**v0.2 实现路径**：
1. 走 **Agnes 图生视频 / SVD 类模型**（如果 Agnes 平台有）
2. 或本机跑 **AnimateDiff / SD-Turbo + ControlNet（pose 序列）**
3. 兜底：1 张 keyframe 让平台运动模板做循环（最低成本上架）

### 2.3 品类 3：LUT 滤镜（推荐 v0.2 备选）

**事实**：
- ✅ 制作最简单：1 个 .cube 文件，几 KB
- ✅ 审核最宽松（不涉及人脸 / 内容判定）
- 🔵 流量稳定但天花板不高（用户更喜欢"看得见的特效"）

**AI 在这里的角色**：
- 🔵 用 **风格化模型（neural style transfer）**生成"参考调色片"，然后用工具反算 LUT
- 🔵 或者用 **Diffusion 重绘多张同场景图**，用色彩对比反推 LUT 矩阵

**为什么列为备选不是首选**：
- 🔵 LUT 的"独特性"很难被用户感知（用户知道是"复古滤镜"但不知道是你做的复古滤镜）
- 🔵 不容易形成账号品牌

## 3. 个人创作者起步局势（你当前位置）

### 3.1 你的真实起点

**事实**：
- ✅ 个人抖音号（无大粉丝基础）
- ✅ 无 MCN 资源、无字节内部联系
- ✅ 1 人开发 + 设计 + 运营
- ✅ 工具具备 AI 出图能力

🔵 **你比 99% 抖音特效新创作者强的地方**：你有完整的 AI 工具链（其他人多数靠 PS / AE 手工），出图速度快 10 倍。

🔵 **你比专业特效创作者弱的地方**：
- 没有 EC 操作经验（其他人摸了 1-2 年）
- 没看过几百个上架案例（不知道什么样的特效会火）
- 没建立账号品牌（没有"看到这个画风我知道是 XX 老师"的辨识度）

### 3.2 起步 4 周路径建议（🔵 完全是我的产品判断）

**Week 1：先上架 1 个简单特效（任何品类都行，目标是走通流程）**
- 用 v0.1 工具出 1 个静态面部贴图
- 主题随便，**目标只有一个**：完整跑通"出图 → 拖进 EC → 提审 → 上架"
- 完成后你才真正知道每一步的细节

**Week 2：转向动态贴纸（S2 品类）**
- 工具 v0.2 加图生序列帧能力
- 选一个**节日 / 时事主题**（如六月毕业季、夏至、世界杯期间等），这种主题平台有流量倾斜
- 出 1-2 个动态头饰特效

**Week 3：建立账号风格**
- 选一个**视觉锚点**（统一画风：赛博 / 国潮 / 油画 / 像素），所有特效保持这个风格
- 这是建立品牌的最低成本方式

**Week 4：复盘**
- 数据：上架数、审核通过率、使用次数、点赞数
- 选**使用次数最高**的那个特效，分析它为什么火
- 决定是继续这个风格还是换方向

## 4. 给工具开发的反推（关键）

基于上面的分析，**工具的优先级应该是这样**：

| 工具能力 | 为什么 | 优先级 |
|---|---|---|
| 静态贴图（S1） | 跑通流程用，不是主战场 | v0.1 必做 |
| **动态贴纸 / 序列帧（S2）** | **未来 80% 价值在这里** | **v0.2 必做** |
| LUT 生成（S3） | 备选品类 | v0.3 选做 |
| Prompt 风格预设 | 帮你建立账号风格统一性 | v0.2 必做 |
| 节日 / IP 模板库 | 节日窗口期一键起特效 | v0.3 必做 |
| 一键提交 EC 工程模板 | 节省 80% 的 EC 操作时间 | v0.3 必做 |
| 数据回流（监控自己上架特效的使用次数） | 知道哪个特效火、哪个不火 | v0.4 选做 |

> **本企划书的核心结论**：v0.1 工具用完后**立刻转向 v0.2 动态贴纸**，不要在静态贴图上做太多。

## 6. 我（作为 PM）对你的几个直接建议

1. **不要追求一发即火**：第一周目标是"上架 1 个不被驳回"，不是"火 1 个"。新创作者第一个特效平均使用次数 < 50，正常
2. **建立你自己的账号特色**：选 1 个画风坚持做 5-10 个特效，比 5-10 个不同画风各做 1 个有用得多
3. **关注创作者中心数据**：上架后**每天看一次数据**，看 3-5 天就知道这个方向有没有戏
4. **不要把工具开发和上架混做**：v0.1 工具已经够了，**先去用，再回来改**
5. **节日窗口比平时重要 10 倍**：六月底是高考结束 + 毕业季，七月是暑假 + 七夕预热，把这些日历钉死
6. **不期望 ROI**：抖音特效个人创作者收益分成对绝大多数人来说是 0-100 元/月级别（🔵 我的判断，待你后台确认），主要价值在**作品 + 经验 + 视觉品牌**
