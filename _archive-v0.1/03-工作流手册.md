# 03 - 工作流手册

> Project DOUYIN-FX ｜ v0.1
> 这是给"我自己"用的操作手册。

## 0. 一图概览

```
[Step 0]                [Step 1-3]                  [Step 4-6]                [Step 7]
EC 准备工作      →    工具产素材包         →     EC 里搭工程              →   抖音提审上架
（一次性）            （每个特效都做一遍）           （每个特效都做一遍）         （每个特效都做一遍）
```

工具只覆盖 Step 1-3，**Step 0 和 4-7 是手工操作，但同样关键**。

---

## Step 0：环境一次性准备

### 0.1 装 Effect Creator

1. 浏览器打开 https://effect.douyin.com → 下载 Effect Creator（macOS / Windows 都有）
2. 用抖音号登录
3. 在创作者中心确认有"特效创作者"权限（个人号默认有）

### 0.2 摸清 EC 资产规范（半天，必做）

不要跳过这一步，否则工具产出的素材在 EC 里用不了。

- 打开 EC，新建一个空项目（File → New → Face Effect 模板）
- 看左侧资产面板支持哪些类型（贴图 / 序列帧 / LUT / 3D 模型）
- 找一个已有官方示例特效，**用 EC 自带的"导出资产"功能**导出一遍，看：
  - 贴图分辨率（一般 512×512 或 1024×1024）
  - 文件命名（一般 ASCII 无空格）
  - 是否带 alpha
  - face mesh 的 UV 模板长什么样
- 把这些信息记到 `04-EC调研笔记.md`

### 0.3 把 EC 的脸部 UV 模板导出

1. 在 EC 的某个 face mesh 节点上找到"展示 UV 网格"
2. 截图或导出 → 保存为 `assets/face_uv_template_v1.png`，1024×1024
3. 这张图就是工具内置的 `face_uv_template_v1`，prompt 引导和后期对齐都依赖它

### 0.4 装工具

```bash
git clone <repo> && cd douyin-fx
pip install -e .

# 设环境变量（如果用远程后端）
export DOUYINFX_REMOTE_API_KEY="sk-xxx"

# 启动
python -m douyinfx
```

### 0.5 模型下载

```bash
python -m douyinfx download-model sdxl-turbo
```

国内拉不动就：

- 改用 hf-mirror.com（命令内置 fallback）
- 或手动从 ModelScope 下载放到 `~/.cache/huggingface/hub/models--stabilityai--sdxl-turbo/`

## Step 1：构思一个特效

举例：「赛博朋克金属面具（S1）」

写一个 prompt 草稿，越具体越好：

```
cyberpunk metal mask, frontal symmetric, glowing cyan circuits,
black and chrome, on transparent background,
aligned to face uv template,
high detail, 8k, no shadow, no background
```

Negative prompt：

```
lowres, blurry, asymmetric, watermark, text, logo,
realistic human face, photo of celebrity, child
```

**Negative prompt 里至少包含 `realistic human face, photo of celebrity, child`**，作为对自己的第一道保护，避免误生成红线内容。

---

## Step 2：在工具里出图

1. 浏览器打开 `http://127.0.0.1:7860`
2. 选 "S1: 面部贴图" tab
3. 后端选 `local_sdxl_turbo`（M1 Pro 16GB），跑不动再换 `remote_openai`
4. 填 prompt / negative
5. seed 留 -1（随机），出几张挑一张满意的；满意后**记下那个 seed**，下次微调能复刻
6. steps：SDXL Turbo 用 4，跑得快效果也够
7. size：1024×1024
8. 点"生成"，等 5-8s

不满意就改 prompt，重复几次。SDXL Turbo 单图 5-8s，迭代成本低。

---

## Step 3：抠背景 + 导出

1. 在 UI 里点"抠透明背景"，rembg 自动出 RGBA 图
2. 检查边缘是否干净（毛糙的话，先改 prompt 让背景更纯色再回来抠）
3. 填项目名（如 `cyberpunk-mask-v1`）
4. 勾选合规自检 7 条 checklist
5. 点"导出素材包 zip"

产物在 `~/douyin-fx-projects/2026-06-10-cyberpunk-mask-v1/`：

```
2026-06-10-cyberpunk-mask-v1/
├── raw/face_paint_main_raw.png
├── processed/face_paint_main.png        # 抠完 + 1024×1024 + RGBA
├── meta.json                            # 完整元数据
└── cyberpunk-mask-v1_pack.zip           # 给 EC 用的素材包
```

---

## Step 4：在 EC 里搭工程

1. EC 里 New → Face Effect 模板
2. 把 `processed/face_paint_main.png` 拖到资产面板
3. 在节点视图里：
   - 找到 face mesh / face tracker 节点
   - 加一个 face texture 组件，绑定刚才的贴图
   - 调透明度 / 混合模式 → 直到看起来和脸贴合
4. 用 EC 内置预览看效果，调到满意
5. （可选）加触发器：张嘴 / 眨眼 → 贴图变化
6. 用 EC 内置录制录一段 demo

具体节点连法看 EC 官方文档，工具不替代这一步。

---

## Step 5：本地真机预览

1. EC 有"扫码到手机抖音"的功能（手机抖音扫 EC 里的二维码）
2. 在手机上看真机效果
3. 真机和 EC 内置预览经常有差异（光照、性能），以真机为准

---

## Step 6：提交审核

1. EC 里 → 发布 → 填特效信息：
   - 特效名（中文，简短）
   - 特效分类
   - **AI 生成标识**：勾选"AI 生成"或在描述里写明（按抖音当时的 UI 指引）
   - 描述
   - 封面图（用 raw/ 里的图自己加文字做）
2. 提交审核
3. 等抖音审核（一般 1-3 天）

---

## Step 7：上架后追踪

1. 抖音创作者中心查看：
   - 审核状态（通过 / 驳回 + 原因）
   - 上架后使用次数 / 投稿数
   - 是否有分成（看品类）
2. 把审核结果（特别是"驳回"的原因）记到 `04-EC调研笔记.md`，下次规避

## 常见问题

### Q：抠完背景边缘毛糙
- 先调 prompt 让背景纯黑或纯白 → 再 rembg 效果更好
- 或在 Photoshop / GIMP / Affinity 里手工修一下边缘

### Q：贴图贴到脸上对不齐
- 出图时 prompt 里强调 `aligned to face uv template, frontal, symmetric`
- 必要时下次直接用 `assets/face_uv_template_v1.png` 作为 img2img 的输入条件（v0.3 引入 ControlNet 后效果会更好）

### Q：mps 跑 SDXL Turbo OOM
- 把 size 调到 768×768
- 或换远程后端

### Q：模型下载失败
- 见 §0.5

### Q：审核被驳回
- 抖音审核理由优先级：素材内容 > AI 标识 > 命名规范
- 内容驳回 → 改 prompt 重做
- AI 标识驳回 → 提审时按抖音模板填全标识字段

### Q：要做序列帧 / LUT
- v0.1 不支持，等 v0.2
- 临时手工：v0.1 工具出 N 张图后用 ffmpeg 拼成序列帧
